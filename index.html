How do I use this?
  • Encode - This will display an emoji for every letter entered.
  • Translate - This will attempt to find an emoji that matches any words entered.
  • Madlib - This will display a random emoji, if one exists, that is similar to any words entered.
  • Search - This will display any & all emojis, that match the words entered.
  • Random - This will choose a random translate method for any words entered.
